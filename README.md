# دراستي

مشروع عربي جاهز للنشر لمحاكاة امتحانات التوجيهي بواجهة حديثة ومتجاوبة، مع ربط مباشر إلى:

- Google Sheets لحفظ النتائج
- Google Drive لتخزين ملفات الامتحانات JSON
- Google Apps Script Web App كطبقة API
- GitHub Pages أو أي استضافة ثابتة للواجهة

## محتويات الحزمة

- `index.html` الواجهة الكاملة
- `apps-script/Code.gs` كود الخادم
- `apps-script/appsscript.json` إعدادات Apps Script
- `sample-exam.json` مثال جاهز للرفع
- `README.md` هذا الملف

## ماذا أصبح جاهزًا في هذه النسخة

- تصميم حديث واحترافي باسم **دراستي**
- متجاوب مع Android وiPhone وPC عبر المتصفح
- لوحة دخول أنيقة
- لوحة طالب حديثة مع بطاقات وإحصائيات وبحث
- شاشة امتحان محسنة مع مؤقت واضح وتقدم الإجابات
- تصحيح فوري للاختياري
- عرض الإجابة النموذجية للمقالي
- تجميع نقاط الضعف
- نتائج الطالب
- لوحة إدارة أجمل مع:
  - تسجيل دخول للإدارة
  - رفع JSON مباشرة
  - استيراد JSON من ملف
  - بحث داخل نتائج الطلاب
- دعم KaTeX للمعادلات
- دعم الوضع الليلي
- تنبيهات مرئية داخل التطبيق
- شريط يوضح حالة الجاهزية للربط

## ما الذي يبقى عليك فقط

### 1) إعداد Google Sheet

- أنشئ ملف Google Sheets جديد
- سمّه مثلًا: `DerastiDB`
- افتح: **Extensions > Apps Script**

### 2) إضافة ملفات Apps Script

- الصق محتوى `apps-script/Code.gs`
- أضف ملف `appsscript.json` والصق محتواه

### 3) نشر Apps Script

من Apps Script:

- **Deploy**
- **New deployment**
- النوع: **Web app**
- **Execute as:** Me
- **Who has access:** Anyone

بعدها انسخ رابط الـ Web App.

### 4) ربط الواجهة

افتح `index.html` وابحث عن:

```js
API_BASE: 'PUT_YOUR_APPS_SCRIPT_WEBAPP_URL_HERE'
```

واستبدله برابط Web App الحقيقي.

### 5) رفع الواجهة

ارفع `index.html` إلى GitHub Pages أو أي استضافة ثابتة.

## ملاحظات مهمة

- المشروع **جاهز للربط** لكنه لا يمكن أن يكون متصلًا فعليًا قبل أن تضع أنت رابط Google Apps Script الحقيقي.
- كلمة مرور الإدارة الحالية هي `admin` داخل الواجهة، وهذا مناسب فقط للمشروع المغلق.
- لو أردت حماية أقوى لاحقًا، اجعل التحقق الحقيقي داخل Apps Script بدل الواجهة.

## توصية تشغيل سريعة

2. خذ الرابط
3. ضعه داخل `index.html`
4. افتح لوحة الإدارة
5. ارفع `sample-exam.json`
6. ابدأ الاستخدام
